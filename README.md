# form（お問い合わせフォーム）
PHPによるお問い合わせフォームです。
MAMPのローカル環境内で作成しました。
フロントエンドにBootstrapを使用しております。
画面遷移ありで、確認画面（check.php）で送信ボタンをクリックするとDBに接続され、
入力フォームのデータはDBへと保存されます。
